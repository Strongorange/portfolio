### AI 여행 플래너와 여행 일정 공유 커뮤니티

> 사용자가 지정한 지역, 기간, 테마에 따라 맞춤형 여행 코스를 제안하고, 사용자가 자신의 여행 코스를 직접 생성하고 공유할 수 있는  
> 플랫폼 서비스입니다.
> </br>  
> **역할**
>
> - 웹, 앱 프론트엔드 작업
> - React Native 사용해 WebView 기반 하이브리드 앱 제작
> - 프로젝트 진행 관리 및 전달
>
>   **개발 기간**: 2022.10 ~ 2022.12
>
> **사용 기술**
>
> - Typescript
> - Vue.js
> - React Native
> - Google Sheets, Figma, Notion
>
> **작업 내용**
>
> - 모바일 페이지 UI/UX 개선
>   - 페이지의 상단부터 시작하는 요소들을 Thumb zone 안으로 이동 제안
>   - PC에서 세로로 스크롤 되는 공지, select로 된 카테고리 를 Carousel 형식으로 가로 스크롤로 변경
> - 현재 위치 기반 추천 여행코스, 주변 여행지 요청
> - WYSIWYG 에디터 라이브러리 react-draft-wysiwyg 사용해 추천, 관리자 페이지 CRUD
> - IOS -Ad hoc 배포 테스트
>
> **문제와 해결**
>
> - Vue.js 사용경험 없이 프로젝트 참여
>
>   - Vue.js 공식 문서를 통해 뷰 디렉티브등 학습
>   - 학습시 막히는 부분 기존 프론트 개발자분과 적극적으로 소통하여 해결
>   - Vuex를 적용하여 전역적으로 사용되는 상태값 관리
>
> - WebView 버전 호환성 문제
>   - @support 규칙을 사용해 지원 브라우저, 비지원 브라우저에 따라 스타일링 적용
>   - 개발시 에뮬레이터만이 아닌 실제 기기에서 테스트 동시에 진행
